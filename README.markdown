# Basic config files

These are configuration files to set up a basic UNIX system with:

* vim
* tmux
* zsh
* rbenv
* git

## Installation

```sh
$ git clone git@github.com:jhbabon/dotfiles.git ~/.dotfiles
$ cd ~/.dotfiles
$ rake install
```
